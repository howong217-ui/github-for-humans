WHAT THIS DOES AND DOES NOT DO

Facts before verdicts.

The current MVP compares repository age, recent pushes, fork behavior, and star velocity. A result moves only when a visible rule applies.

It does not decide whether a repository is fake, safe, human-written, or high quality. High growth can be legitimate and low forks can be normal. The evidence is a prompt to inspect, not a substitute for judgment.
